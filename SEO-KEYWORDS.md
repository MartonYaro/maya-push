# MAYA Push — семантическое ядро (SEO / ASO)

Рынки: **RU**, **Europe (EN)**, **USA**. Продукт: мотивированные установки + ASO
для **App Store и Google Play**.

> ⚠️ Объёмы запросов здесь не проставлены — их надо валидировать в инструменте
> (Asodesk / Mobile Action / AppTweak / Keyword Tool / Google Keyword Planner /
> Yandex Wordstat для RU) перед тем, как закладывать в контент-план. Ниже —
> сгруппированное по интенту ядро и приоритет на основе коммерческого смысла.

Легенда приоритета: 🔴 деньги (high-intent, в title/H1/H2) · 🟡 средний
(разделы, FAQ) · ⚪ верх воронки (блог).

|---|---|
| `<title>` / H1 | 🔴 ядро: «мотивированные установки», «ASO», «App Store **и Google Play**» |
| H2 секций | «ASO-оптимизация», «продвижение приложений в App Store и Google Play», «продвижение по ключевым словам» |

| FAQ | инфо-ключи + перехват «накрутки» в белом ключе + «iOS и Android / App Store и Google Play» |
| Блог (на будущее) | ⚪ верх воронки: «как продвинуть приложение…», «что такое ASO», «ASO vs SEO» |

## Структурные задачи под рынки
- **EN-рынок (EU/USA):** нужна отдельная индексируемая англ. страница (текст в
  HTML, не в `data-en`-тогле) + `hreflang` (ru / en / x-default). Иначе по
  английским ключам ранжирования не будет.
- **Google Play:** сейчас лендинг только про iOS — добавить Android/Google Play
  в hero, фичи и FAQ (поддержка в продукте уже есть).
- **Блог/контент-хаб** под ⚪-кластер — основной источник органики на дистанции
  (по коммерческим ключам конкуренция высокая, инфо-запросы заходят быстрее).
